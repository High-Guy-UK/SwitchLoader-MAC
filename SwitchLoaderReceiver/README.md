# SwitchLoader JS Prototype

This folder contains an nx.js visual prototype for the Switch-side companion experience.

The production app is developed primarily in the native companion target, while this prototype remains useful for UI ideas and interaction experiments.

## Prototype Build

Install Node.js, then from this folder:

```sh
npm install
npm run build
```
